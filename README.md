# Facial Recognition Access System

## 🚀 Overview
This project implements a real-time facial recognition system using Python and OpenCV.

The application captures video from a webcam, detects human faces, and performs facial recognition using the LBPH (Local Binary Patterns Histograms) algorithm.

The system was developed as part of a Bachelor's Thesis in Telecommunications Engineering focused on biometric authentication systems.

---

## 🧠 Features
- Real-time face detection
- Facial recognition system with OpenCV
- Webcam integration
- LBPH face recognition model
- Lightweight and modular architecture
- Easy to run and extend

---

## 🛠 Tech Stack
- Python
- OpenCV
- NumPy
